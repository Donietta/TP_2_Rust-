# Programme en Logo pour dessiner un carré #
Pour dessiner un carré en respectant la syntaxe logo proposée, on peut utiliser la commande repeat et un bloc d'instructions.
On répète 4 fois l'instruction "avancer de 100 puis tourner à 90°" afin de former un carré : 


$repeat 4 [
  forward 100
  right 90
]$
